# Angular 动态化加载脚本demo

在实际的开发中,由于功能众多产生的脚本文件也随之增多.而Angular本身并不能动态化的加载脚本.市面上动态加载器很多,
比如require,但是相对来说require还是重了一点,对angular来讲也不是太完美.所以,一个为angular量身定制的脚本加载器就出现了,
它就是`ocLazyLoad`


## Getting Started

执行: `bower install`
